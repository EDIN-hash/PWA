🆕 Новые функции в версии 2.0.3

1. Скрытая роль "Moderator" (moder)
   - Назначается вручную через SQL
   - Имеет все права админа:
     - Может добавлять новые предметы
     - Может редактировать существующие предметы
     - Может удалять предметы
   - Имеет доступ к функции генерации ID
   - Не доступна при регистрации

2. Функция "Get Free ID" (только для moder)
   - Генерация уникальных ID для новых предметов
   - Доступна только для роли "moder"
   - Модальное окно с выбором категории
   - Для телевизоров: поле для ввода любого размера экрана
   - Кнопка копирования ID в буфер обмена
   - Умная генерация с учетом существующих ID
   - Автоматическое формирование ID с указанным размером

Форматы ID:
- Телевизоры: TV{размер}{номер} (например, TV55001, TV65001, TV100001)
- Холодильники: L{номер} (L001, L002)
- Экспрессо: E{номер} (E001, E002)
- Стулья: K{номер} (K001, K002)
- Инструменты: NM{номер} (NM001, NM002)
- LADY: A{номер} (A001, A002)

- Проблема: Кнопки слишком маленькие на мобильных
  Решение: Используются адаптивные размеры с классами w-full на мобильных
- Проблема: Поле поиска неудобно использовать
  Решение: На мобильных поле занимает всю ширину для удобного ввода
- Проблема: Модальные окна выходят за пределы экрана
  Решение: Используются адаптивные модальные окна с max-w-none
- Проблема: Кнопки накладываются друг на друга
  Решение: Используется flex-direction: column на мобильных
- Проблема: Текст слишком мелкий
  Решение: Используются адаптивные размеры шрифтов

📊 Сортировка товаров

Схема сортировки:
- По умолчанию: сортировка по ID (имени) в алфавитном порядке
  - Пример: TV55001, TV55002, TV65001, L001, L002
  - Это обеспечивает естественный порядок товаров по их номерам
- Доступные параметры сортировки (для админов и модераторов):
  1. Wysokość (высота) - по полю wysokosc
  2. Szerokość (ширина) - по полю szerokosc
  3. Głębokość (глубина) - по полю glebokosc
  4. Ilość (количество) - по полю ilosc
- Направление сортировки:
  - По умолчанию: по возрастанию (asc)
  - При повторном нажатии: по убыванию (desc)
  - При нажатии "Reset": возврат к сортировке по умолчанию (по ID)
- Алгоритм сортировки:
  - Сравниваются числовые значения полей
  - Поддерживает польский формат чисел (запятая вместо точки)
  - Преобразует значения в числа перед сравнением
  - Если значение отсутствует, используется 0
  - Возвращает -1, 0 или 1 для сравнения
  - Для сортировки по умолчанию используется localeCompare для строк
- Визуальная индикация:
  - Стрелка ↑ для сортировки по возрастанию
  - Стрелка ↓ для сортировки по убыванию
  - Нет стрелки для сортировки по умолчанию (по ID)
- Пример:
  - По умолчанию: TV55001, TV55002, TV65001 (сортировка по ID)
  - Нажмите "Wysokość" - сортировка по высоте по возрастанию
  - Нажмите "Wysokość" снова - сортировка по высоте по убыванию
  - Нажмите "Reset" - возврат к сортировке по ID
- Доступ к сортировке:
  - Админы и модераторы могут использовать все параметры сортировки
  - Обычные пользователи видят товары в порядке по умолчанию
- Исправление сортировки:
  - Ранее высота и ширина не сортировались правильно
  - Проблема была в польском формате чисел (запятая вместо точки)
  - Теперь все параметры сортируются корректно

🎯 Как использовать:
1. Создайте пользователя с ролью "moder" через SQL:
   INSERT INTO users (username, password, role) VALUES ('moder1', 'pass', 'moder');
2. Авторизуйтесь как модератор
3. Теперь у вас есть все права админа:
   - Добавление новых предметов (кнопка "Add Item")
   - Редактирование существующих предметов (кнопка "Edytuj" на карточках)
   - Удаление предметов (кнопка "Usuń" на карточках)
   - Просмотр всех предметов
4. Дополнительно у вас есть доступ к функции "Get Free ID":
   - Нажмите "Get Free ID" в верхней панели
   - Выберите категорию:
     - Для телевизоров: введите любой размер экрана (например, 55, 65, 100)
     - Для других категорий: поле размера не отображается
   - Нажмите "Get Free ID" для генерации
   - Пример работы:
     - Если в базе есть TV80060, следующий ID будет TV80001 (первый свободный)
     - Если в базе есть TV55001 и TV55003, следующий ID будет TV55002
   - Если все ID для указанного размера заняты (999 штук):
     - Вы увидите сообщение об ошибке
     - Попробуйте другой размер или обратитесь к администратору
   - Скопируйте сгенерированный ID
   - Используйте ID при добавлении нового предмета

- Это означает, что товары будут отображаться в порядке: TV55001, TV55002, TV65001, L001, L002
- Такой порядок обеспечивает естественную сортировку по номерам товаров
- Вы можете изменить сортировку, нажав на кнопки "Wysokość", "Szerokość", "Głębokość" или "Ilość"
- Чтобы вернуть сортировку по умолчанию, нажмите "Reset"

🔧 Алгоритм:
- Для телевизоров:
  - Использует любой указанный размер экрана (например, 55, 65, 100)
  - Проверяет существующие ID с указанным размером (например, TV80%)
  - Находит первый свободный номер с 1 до 999
  - Формат: TV{размер}{номер} (например, TV55001, TV100001)
  - Максимально: 999 ID на каждый размер
  - Пример: если есть TV80060, следующий ID будет TV80001 (а не TV80061)
- Для других категорий:
  - Проверяет существующие ID с соответствующим префиксом
  - Находит первый свободный номер
  - Формат: {префикс}{номер} (например, L001, E001)
- Гарантирует уникальность в пределах категории
- Выдает понятное сообщение об ошибке, если все ID для указанного размера заняты
